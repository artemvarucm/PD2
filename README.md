### 🚀 Ejecutar el proyecto
#### Requisitos
+ `python` >= 3.12
+ `uv` >= 0.4
#### Ejecución
Instalación de librerías y activación del entorno

```bash
$ uv sync
$ source .venv/bin/activate
```
Ejecución del notebook seleccionando el entorno virtual seleccionado
